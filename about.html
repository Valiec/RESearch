About RESearch:

RESearch is a web tool which searches long DNA sequences for restriction enzyme sites.

This tool uses a data structure known as a trie, containing the regognition sequences of 600+ commercially available restriction enzymes. Specifically, when the page is loaded, it constructs the trie from a list of restriction enzyme sequences, since the trie is large enough that downloading the trie itself would likely slow down page loading.

Once the trie is constructed, it uses the search sequence entered to traverse the trie and find the matching restriction sites.

What is a Trie?

A trie is a variety of a data structure known as a tree, where each node is a single character in one or more of the patterns used to make the trie, and the child node or nodes of any node in the trie are each a character which follows their parent node's character in one or more of the patterns. The root node is a special case. The characters of the child nodes of the root node are simply the first characters of the patterns.
